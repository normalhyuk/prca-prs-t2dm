# Penn Medicine BioBank: Sample Collection SOP

**References**:
- [PMBB Data Access](https://pmbb.med.upenn.edu/data-access/index.php)
- [PMBB Protocol Paper](https://www.mdpi.com/2075-4426/12/12/1974)
- [PMBB Consent Form v12](https://pmbb.med.upenn.edu/documents/informed_consent_v12.pdf)

## Procedure [From PMBB Protocol]
1. **Recruitment**: EMR pre-visit check-in (myPennMedicine portal)
2. **Sample**: 6mL EDTA tube (purple top) via outpatient phlebotomy
3. **Consent**: 3-page electronic form + CRC support (7:30AM-5PM)
4. **Eligibility**: Age ≥18, Penn Medicine patient, consent capacity
5. **Volume Check**: >5mL required for processing
