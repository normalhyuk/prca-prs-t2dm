# Penn Medicine BioBank: Sample Processing SOP

**References**:
- [PMBB Genomics Methods](https://www.mdpi.com/2075-4426/12/12/1974)
- [Ritchie Lab Pre-Imputation QC](https://ritchielab.org/pmbb-imputation-docs/#pre-imputation-qc-on-study-data)

## Procedure
1. **LIMS Scan**: Barcode verification upon receipt
2. **Centrifuge**: 4000rpm, 10min, 4°C
3. **Aliquot**:
   - Plasma: 500μL × 2 (-80°C)
   - Buffy coat: 1mL (DNA extraction)
4. **QC Metrics**: Hemolysis index <500, volume verification
5. **Storage**: -80°C within 6h
