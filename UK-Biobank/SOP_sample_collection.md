# UK Biobank: Sample Collection SOP

**References**: 
- [UKB Data Access Guidelines](https://www.ukbiobank.ac.uk/about-us/how-we-work/access-to-uk-biobank-data/)
- [UK Biobank Blood Manual](https://biobank.ndph.ox.ac.uk/ukb/ukb/docs/Bloodsample.pdf)

## Procedure
1. **Baseline/Imaging Visit**: 50mL EDTA blood (5x 10mL Vacutainers)
2. **Consent**: Electronic informed consent via UKB portal
3. **Collection**: Phlebotomy within 2h processing window
4. **Labeling**: UKB_[eid]_[visit]_[matrix]
5. **Transport**: 4°C to central processing facility
